# Minecraft-Modpack

## Requirements

- Minecraft Forge Version 1.18.1-39.0.64+ [Download](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.18.1.html)
- 4GB+ RAM (-Xmx4G in den startparametern)
